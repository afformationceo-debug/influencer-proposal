How to Use
3단계로 시작하세요
복잡한 설정 없이 키워드만 입력하면 자동으로 인플루언서를 수집합니다.
01
🔍

키워드 입력

수집하고 싶은 키워드를 입력합니다. 플랫폼별로 최적화된 검색이 자동으로 실행됩니다. 국가, 목표 수량도 설정 가능합니다.

02
⚡

자동 수집

Start 버튼을 누르면 시스템이 자동으로 프로필을 방문하고 이메일, 팔로워 수, 계정 유형 등을 수집합니다. 실시간 진행률을 확인하세요.

03
📥

데이터 활용

수집된 인플루언서 목록을 CSV로 내보내 마케팅에 바로 활용하세요. 이메일, 팔로워 수, 국가별 필터링이 가능합니다.

Platform Features
주요 기능
3개 플랫폼 통합 관리, AI 기반 자동 분류, 실시간 수집 모니터링을 제공합니다.
📱

3개 플랫폼 통합

Instagram, TikTok, Twitter(X)를 하나의 대시보드에서 관리합니다. 플랫폼별 특화된 수집 로직이 내장되어 있습니다.

🤖

AI 자동 분류

수집된 계정을 AI가 자동으로 인플루언서/비즈니스 계정으로 분류합니다. 비즈니스 계정은 자동 제외되어 순수 인플루언서만 확보합니다.

📧

이메일 자동 추출

프로필 소개글에서 이메일을 자동으로 추출합니다. 이메일이 있는 인플루언서만 필터링하여 즉시 아웃리치가 가능합니다.

🌍

국가별 필터링

AI가 프로필 내용을 분석하여 국가를 자동 판별합니다. 미국, 일본, 대만, 태국, 베트남 등 글로벌 시장별 인플루언서를 분리 관리합니다.

📊

실시간 모니터링

수집 진행 상황을 실시간으로 확인합니다. 현재 수집 중인 키워드, 진행률, 성공/스킵 건수를 대시보드에서 바로 볼 수 있습니다.

💾

CSV 내보내기

수집된 데이터를 언제든 CSV 파일로 내보낼 수 있습니다. 전체 또는 필터링된 결과를 선택적으로 다운로드합니다.

📧

이메일 자동 발송 시스템

수집된 이메일로 자동 발송. 템플릿 + 개인화 변수(이름·국가·관심사), SMTP/SendGrid 연동, 열람·클릭·응답 트래킹. A/B 테스트 지원으로 발송 성과 최적화.

📱

Instagram DM 자동 발송

수집된 인플루언서에게 Instagram DM 자동 송출. 계정 차단 방지 스마트 스로틀링(시간대별 분산), 다중 계정 로테이션, 응답자 자동 분류로 CRM 전환 준비까지 완료.

📊

응답 추적 & 리포트

이메일/DM 발송 후 열람·응답률 실시간 대시보드. 주간·월간 성과 리포트 자동 생성. 전환율 높은 메시지 템플릿 제안.
